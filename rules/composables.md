# @leitware/composables-cli AI Usage Rules

Generated by `bun scripts/generate-rules.ts`. Do not hand-edit this file.

## Package Rules

- Install with `npm install @leitware/composables-cli`.
- Import normal components, hooks, and utilities from `@leitware/composables-cli`.
- Import AI components from `@leitware/composables-cli/ai`.
- Import `@leitware/composables-cli/styles.css` once in root/global CSS before using components.
- Import at most one preset after the base styles, for example `@leitware/composables-cli/presets/brutalist.css`.
- Prefer `Form*` components when building labeled fields with description, validation, or error text.
- Prefer array APIs where provided, such as `Tabs`, `Breadcrumb`, `Pagination`, `Accordion`, `Carousel`, and form selects.
- Do not import from `src/_internal` or package-internal file paths in downstream apps.
- Use semantic token utilities such as `bg-surface-*`, `text-*`, `border-*`, `stroke-*`, and `icon-*` rather than arbitrary colors.
- Use `cn` from the package when composing class names.

## Required CSS

```css
@import '@leitware/composables-cli/styles.css';

/* Optional: apply exactly one preset after the base styles. */
@import '@leitware/composables-cli/presets/brutalist.css';
```
